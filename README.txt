Présentation du projet : 

Cet exercice a pour objectif de mettre en pratique les notions que vous avez acquises dans les deux premières parties de ce cours sur Git et Githug.

Vous devez créer un repository Git simple contenant : 
1) Un fichier nommé "README.txt" qui présente le projet 
2) Au moins 2 autres fichiers de votre choix 
3) Au moins 4 commits dinstincts montrant l'ajout puis la modification de certains fichiers
